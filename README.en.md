# LibraryManagementSystem-SUST

#### Description
陕西科技大学-电子信息与人工智能学院-物联网192-图书馆管理系统Android端
1.实现用户系统登录、验证、超时归还提醒等功能；2.管理员对图书信息增删改等操作，管理员对图书的管理及权限分配等。3.图书信息查询、借阅登记、归还时间、所在书架位置等；4.手机APP实现用户的登录、查询等简单便捷的功能。
